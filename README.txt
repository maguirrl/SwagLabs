1. Abrir el proyecto con IntelliJ IDEA 
2. Esperar a que cargue todas las dependencias (Se visualiza en la parte inferior del programa como una barra de carga)
3. Dar clic a boton Run para poder correr la automatización (El tag ya esta ingresado en el runner para que solo sea ejecutado)
4. Poder evidenciar la correcta ejecución de este
5. Para visualizar el reporte abrir la carpeta target/cucumber-html-report y abrir index.html (Si no se visualiza correctamente copiar la carpeta cucumber-html-report en el escritorio y desde ahí abrir index.html)
